title: How to export the translation text as an USFM?
---

You export the translation text as a usfm file to use as a backup or to share it with other editors. To export the translated text as a USFM file to the wantd location in your device, follow the steps below.

-   Click on the **Export** menu (refer [_Workspace User Interface_](../../Part-1/Getting-Started/Autographa-Live-User-Interface-Overview/p1-2-2-main-wrkspace-ui.mdx))

_Figure 2.9.1.1_ displays the **Export** menu.

<img src="/AutographaLiveImages/Export-translation-text/export-menu-fig-2.9.1.1.jpg" width="100%" alt="Export menu"/>

<div align='center' style={{ fontStyle: 'Italic' }}>
	Figure 2.9.1.1- Export menu
</div>

-   Select **USFM** option from the **Export** menu

_Figure 2.9.1.2_ displays **Export as USFM** pop up.
<img src="/AutographaLiveImages/Export-translation-text/export-usfm-fig-2.9.1.2.jpg" width="100%" alt="Export as USFM"/> 

<div align='center' style={{ fontStyle: 'Italic' }}>
	Figure 2.9.1.2 - Export as USFM
</div>

-   Enter the stage name of translation
-   Choose **Current Book** or **All Books** option based on your requirement
-   Click **Stage in Translation** menu
-   Select the stage from the menu
-   Click **Export**

_Figure 2.9.1.3_ displays **Export as USFM** pop up window.
<img src="/AutographaLiveImages/Export-translation-text/export-usfm-confirmation-fig-2.9.1.3.jpg" width="100%" alt="Export as USFM"/> 
<div align='center' style={{ fontStyle: 'Italic' }}>
	Figure 2.9.1.3 - Export as USFM confirmation
</div>

-   Click **ok** in **Export as USFM** pop up.
