title: How to remove the reference text?
---

-   Go to **Settings**>**Manage Reference Texts**

_Figure 1.2.1.7_ displays the **Manage Reference Texts** screen and its options.

<img src="/AutographaLiveImages/Getting_Started/manage-reference-texts-fig-1.2.1.7.jpg" width="100%" alt="Manage Reference Texts screen"/>

<div align='center' style={{ fontStyle: 'Italic' }}>Figure 1.2.1.7 - Manage Reference Texts screen</div>

-   Click on **Remove** option under the **Action** column of the required reference text

_Figure 2.3.3.2_ displays the **Confirmation** pop up to confirm the deletion of the reference text.
<img src="/AutographaLiveImages/Settings/confirmation-pop-up-fig-2.3.3.2.jpg" width="100%" alt="Confirmation pop up"/>

<div align='center' style={{ fontStyle: 'Italic' }}>Figure 2.3.3.2 - Confirmation pop up</div>

-   Click **ok**
-   After a few seconds the reference text will be deleted and you come back to the Workspace

:::note
To cancel the deletion of the reference text, click **cancel**.
:::
