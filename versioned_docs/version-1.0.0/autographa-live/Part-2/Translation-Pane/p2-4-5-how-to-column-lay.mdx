To use different column layouts, follow the steps below.

-   Click on **2x** on **Column layout selector** to view a reference text.

_Figure 2.4.5.1_ displays the **2x Column layout**
<img src="/AutographaLiveImages/Translation-pane/2x-column-layout-fig-2.4.5.1.jpg" width="100%" alt="2x Column layout"/>

<div align='center' style={{ fontStyle: 'Italic' }}>Figure 2.4.5.1- 2x Column layout</div>

-   Click on **3x** on **Column layout selector** to view 2 different reference texts.

_Figure 2.4.5.2_ displays the **3x Column layout**.
<img src="/AutographaLiveImages/Translation-pane/3x-column-layout-fig-2.4.5.2.jpg" width="100%" alt="3x Column layout"/> 

<div align='center' style={{ fontStyle: 'Italic' }}>Figure 2.4.5.2 - 3x Column layout</div>

-   Click on **4x on Column layout selector** to view 3 different reference texts.

_Figure 2.4.5.3_ displays the **4x Column layout**.
