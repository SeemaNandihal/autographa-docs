---



To edit the name of a book in the Book and Chapter menu, follow the steps below.

-   Click on **Book and Chapter** menu

Figure 2.2.1.1 displays the **Book** menu

<img src="/AutographaLiveImages/Book_Chapter/book-menu-fig-2.2.1.1.jpg" width="100%" alt="Book menu"/>

<div align='center' style={{ fontStyle: 'Italic' }}>
	Figure 2.2.1.1 - Book menu
</div>


-   Select the book name you want to edit

Figure 2.2.1.2 displays the **Chapter** menu of the selected book.


<img src="/AutographaLiveImages/Book_Chapter/chapter-menu-fig-2.2.1.2.jpg" width="100%" alt="Chapter menu"/>
<div align='center' style={{ fontStyle: 'Italic' }}>
	Figure 2.2.1.2- Chapter menu
</div>


-   Go back to the **Book** menu by clicking on the **Book** button

Figure 2.2.2.1 displays the edit icon beside the book name

<img src="/AutographaLiveImages/Book_Chapter/book-name-edit-icon-fix-2.2.2.1.jpg" width="100%" alt="Book name edit option"/>
<div align='center' style={{ fontStyle: 'Italic' }}>
Figure 2.2.2.1- Book name edit option
</div>

-   Click on the edit icon beside the required book name

Figure 2.2.2.2 displays **Translate Book Name** pop up.

<img src="/AutographaLiveImages/Book_Chapter/translation-book-name-fig-2.2.2.2.jpg" width="100%" alt="Translate Book Name"/>

<div align='center' style={{ fontStyle: 'Italic' }}>
Figure 2.2.2.2- Translate Book Name
</div>

-   Type the new name on the **Translate Book Name** field

Figure 2.2.2.3 displays **Translate Book Name** pop up screen with a new name.

<img src="/AutographaLiveImages/Book_Chapter/saving-translated-book-name-fig-2.2.2.3.jpg" width="100%" alt="Saving translated book name"/>

<div align='center' style={{ fontStyle: 'Italic' }}>
Figure 2.2.2.3- Saving translated book name
</div>


-   Click **save** to save the translated name of the book
