**Translation Details** settings helps you to define the basic information about your translation text and to choose a file location to save your translation text on your device.

_Figure 1.2.1.4_ displays the details of your translation text that you define.
![alt text](/AutographaLiveImages/Getting_Started/translation-details-fig-1.2.1.4.jpg 'Translation Details of the translation text')

<div align='center' style={{ fontStyle: 'Italic' }}
>Figure 1.2.1.4 - Translation Details of the translation text</div>

_Table 1.2.4_ defines each detail that you enter.

| Name                       | Description                                                                                                                                                                                                                                                                                   |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Language Code**          | The menu of International Codes for the language names (ISO 639-2).  <br/> Example: English (eng) <br/> To know the list of language codes, refer https://www.loc.gov/standards/iso639-2/php/code_list.php |
| **Version**                | The Bible version of your translation. <br/> Example: ULT                                                                                                                                                                                                                         |
| **Export Folder Location** | The folder location on your device into which you want to save the translation text file (USFM file).                                                                                                                                                                                         |
| **Script Direction**       | The direction of the text in the **Translation pane**. There are two options for **Script Direction**: <br/> LTR-left to right <br/> RTL-right to left                                                                                                          |
| **Auto-backup**            | To set up the timing for auto saving your translation text in **Export Folder Location**. There are three options for **Auto-backup**: <br/> **None <br/> Daily** <br/> **Weekly**                                            |

<div align='center' style={{ fontStyle: 'Italic' }}
>Table 1.2.4 - Translation Details</div>
