---

**_Autographa Live_** is an open source software and available for free for you to download from www.autographa.org. You use the following steps to download and install **_Autographa Live_**:

-   Go to https://autographa.org
-   Click on the **Download Now** button on the top right of the screen
-   Click on the **Autographa Live** button to download the **_Autographa Live_** application
-   Click on the required version of the **_Autographa Live_** application
-   Based on the operating system used on your device, download the suitable file displayed under **Assets**
* You download the Autographa setup file based on the operating system on your device as given below in _Table 1.1.2_.

|  Operating System (OS)                | **_Autographa Live_** version                                               |
| --------------------------------------|--------------------------------------------------------------------------- |
| Windows OS | <p style={{ fontFamily: 'Corbel'}}>autographa-live.win-x64-_X.X.X_-beta._X_-setup.exe</p> |
| MacOS  | <p style={{ fontFamily: 'Corbel'}}>autographa-live.mac-_X.X.X_-beta._X_.dmg</p>                |
| Ubuntu OS | <p style={{ fontFamily: 'Corbel'}}>autographa-live.amd64-_X.X.X_-beta._X_-setup.deb</p>    |
<div align="center"style="font-style: italic;">Table 1.1.2- Operating Systems and Autographa Live versions</div>

-   Right click on the downloaded file by keeping the cursor on the file
* On the menu option that appears on right click, select **run** 
* Give neccessary permissions requested by your device and **_Autographa Live_** will be installed in your device
-   After installing the application, you use the **_Autographa Live_** shortcut available on the Desktop to access the application

:::note
Some devices may prevent installation of **_Autographa Live_** because of Windows security restrictions. In that case,
if you receive a pop up screen with the message, “Windows protected your PC”, perform the following actions.

-   Click on **More info** button
-   Click **Run anyway** button at the bottom of the window


:::
