From the GitHub repository, the user can obtain reference materials for the Bible, Open Bible Stories, and Audio, as well as import data from local storage.

**To access Resource Collections:**

- Select the Add Icon.
- A tab for “Custom resources” and “Resource collections” will appear.
- Click on the "Resource collections" tab to open a pop-up window and select the resources to download for reference.
- The selected resource appears in the window.
- The user has the option to update the resource to the latest version by clicking on update icon and delete it whenever necessary.

**To access the Custom Resources:**

- Click on the “Custom Resource” tab.
- Select the file to upload from the local device, then upload it to Autographa.
- This resource can then be used as a reference.
